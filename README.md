# Тестовое задание: Группировка строк

Данный проект является решением тестового задания по обработке и группировке строк.

## Задание
Необходимо прочитать файл `lng.txt`, состоящий из строк вида `A1;B1;C1`. 
Если две строки имеют совпадающие непустые значения в одной и той же колонке, они принадлежат одной группе. 
Результат должен быть выведен в файл `out.txt`, отсортированный по размеру групп (от самых больших к меньшим).

**Основные требования:**
- Время выполнения: не более 30 секунд.
- Потребление памяти: не более 1 Гб.
- Обработка строк с любым количеством колонок.
- Исключение некорректных строк

## Использованные технологии
- Java 21
- Maven

## Сборка проекта
Для создания JAR-файла выполните:
```bash
mvn clean package
```

## Запуск
Программа принимает путь к входному файлу в качестве аргумента:
```bash
java -Xmx1G -jar target/uno-1.0-SNAPSHOT.jar lng.txt
```
Результат будет сохранен в файл `out.txt` в папке запуска.
